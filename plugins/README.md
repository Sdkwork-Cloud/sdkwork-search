# plugins/

Application/runtime plugin source packages, marketplace plugin implementations, or extension packages.

## Purpose

Reserved for application/runtime installable extension source. Currently inactive for `sdkwork-search`.

## Owner

`sdkwork-search` workspace maintainers.

## Allowed Content

- Plugin source packages
- Marketplace plugin implementations
- Extension packages

## Forbidden Content

- Agent plugins (belongs in `.sdkwork/plugins/`)
- Generated SDK output
- Live secrets or credentials

## Related Specs

- `../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md`

## Verification

No active plugins currently. Activate when application/runtime plugins are authored.
