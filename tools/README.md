# tools/

Developer, validation, generation, migration, and operator tools that are not shipped as app runtime code.

## Purpose

Reserved for reusable validators, generators, migration tools, parsers, CLIs, and operator utilities. Currently inactive for `sdkwork-search`.

## Owner

`sdkwork-search` workspace maintainers.

## Allowed Content

- Validators
- Generators
- Migration tools
- Parsers
- CLIs
- Operator utilities

## Forbidden Content

- App runtime code
- Generated SDK output
- Live secrets or credentials
