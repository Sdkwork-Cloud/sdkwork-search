# Plugins

Application-local agent extensions for SDKWork Search Flutter Mobile.

## Adding Plugins

1. Create a directory: `.sdkwork/plugins/<plugin-name>/`
2. Add `.codex-plugin/plugin.json` for installable plugins
3. Follow root plugin workspace rules
