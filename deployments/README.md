Deployment descriptors, environment topology, packaging handoff files, infrastructure examples, and release deployment documentation.

## Purpose

Reserved for Docker, Kubernetes, systemd, nginx, environment topology, release handoff, or deployment runbook content. Currently inactive for `sdkwork-search`.

## Owner

`sdkwork-search` workspace maintainers.

## Allowed Content

- Docker configurations
- Kubernetes manifests
- Systemd service files
- Nginx configurations
- Environment topology
- Release handoff files
- Deployment runbooks

## Forbidden Content

- Live secrets or private keys
- Local override files
- Runtime user config
